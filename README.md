# my-env

## Prerequisites

* xcode
* brew
* git
* zsh
* nvm

## Installation
  
    cd && git clone git@github.com:bcaudan/my-env.git
    shopt -s dotglob
    mv my-env/* . && rmdir my-env

## ZSH Theme

  https://github.com/robbyrussell/oh-my-zsh  
  
    ln -s ~/etc/bcaudan.zsh-theme ~/.oh-my-zsh/themes/bcaudan.zsh-theme
